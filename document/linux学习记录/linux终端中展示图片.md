1. ### 首先我们要先安装一下cacaview

命令如下：

sudo apt install caca-utils

安装完成后执行：

cacaview 1.jpg

回车！！！，一张字符组成的图片就显示出来了

还可以通过按g/G来调整图片的深浅，按d改变图片Dither，来达到最好的显示效果
